## 📋 Описание проекта

**Гостиница РГПУ** — это полноценный веб-сайт для бронирования номеров в Университетской гостинице им. А.И. Герцена. Проект разработан в рамках курсовой работы и включает в себя:

- 📄 **Публичную часть** — просмотр номеров, услуг, отзывов, контактов.
- 🔐 **Авторизацию и регистрацию** — личный кабинет пользователя.
- 📅 **Бронирование** — выбор дат, автоподсчёт цены, календарь доступности.
- 👤 **Личный профиль** — ФИО, Email, телефон, паспорт, история бронирований.
- 💬 **Отзывы** — отображение отзывов гостей на главной странице.

## 🛠️ Технологии

### Фронтенд
- **React 19.0.0** — библиотека для пользовательских интерфейсов
- **Tailwind CSS 4.0.0** — утилитарный CSS-фреймворк
- **AOS** — анимации при прокрутке
- **React Router** — маршрутизация между страницами
- **React Datepicker** — календарь для выбора дат
- **Lucide React** — иконки в стиле тёмно-синих линий

### Бэкенд
- **ASP.NET Core 9.0** — фреймворк для веб-API
- **Entity Framework Core** — ORM для работы с базами данных
- **SQL Server 2022** — реляционная база данных
- **Swagger** — документация API

---

## 🗂️ Структура базы данных

Проект использует базу данных на SQL Server со следующими таблицами:

| Таблица | Описание |
|---------|----------|
| `Номер` | Информация о номерах (тип, вместимость, цена, фото, удобства) |
| `Пользователь` | Данные для авторизации (логин, пароль, роль) |
| `Гость` | Данные гостей (ФИО, Email, телефон, паспорт) |
| `Бронирование` | Бронирования номеров гостями |
| `Услуга` | Список услуг гостиницы |
| `Заказ_Услуги` | Заказанные услуги для бронирования |
| `Отзыв` | Отзывы гостей о номерах |
